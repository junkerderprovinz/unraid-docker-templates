# Standard Notes Web UI — Unraid template

Community Applications template for the official [Standard Notes](https://standardnotes.com) web client.

- **Template:** [`standardnotes-webui.xml`](standardnotes-webui.xml)
- **Install:** Unraid → **Apps** → search *Standard Notes Web UI*
- **Upstream:** <https://standardnotes.com>
